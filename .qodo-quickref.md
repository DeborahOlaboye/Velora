# Qodo Quick Reference Card

## üöÄ Common Commands

### Code Review
```bash
qodo review                    # Review all changes
qodo review <file>            # Review specific file
qodo review --security        # Security-focused review
```

### Testing
```bash
qodo test                     # Run all tests
qodo test --coverage          # Run with coverage report
qodo test <file>              # Test specific file
```

### Security
```bash
qodo security scan            # Scan for vulnerabilities
qodo gas analyze              # Check gas optimization
qodo audit                    # Full security audit
```

### Documentation
```bash
qodo docs generate            # Generate documentation
qodo docs natspec             # Update NatSpec comments
```

### Custom Project Commands
```bash
qodo run build                # Build smart contracts
qodo run test                 # Run contract tests
qodo run deploy               # Deploy contracts
qodo run dev                  # Start frontend dev
qodo run lint                 # Run linting
```
